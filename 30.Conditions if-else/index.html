Conditions

In javascript we have the following conditions statements:

  • a) Use if to specify a block of code to be executed, is a specified condition is true
  • b) Use else to specify a block of code to be executed, if a same condition is false
  • c) Use if else to specify a new condition to test, if the first condition is false
  • d) use switch to specify many alternative blocks of a code to be executed

Conditions if:

The if statement is the fundamental control statement that allows javascript to make decisions & executed statement conditionally

  • if(condition){
    //block off a code to be executed if the condition is true
    }

EXAMPLES

let age = 12;
if (age > 9 ){ console.log("Age = "+age); }

Conditions else:
