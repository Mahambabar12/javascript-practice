Case sensitive & camel sensitive

Case Sensitive

01. Variable are case sensitive
02.So rose and Rose are two different variables

  • var rose = "Hello;
  • var Rose = "Hello";
  • alert (rose);
  • alert (Rose);
  • alert(ROSE); //Error

Camel case

01. If there more than one word in the variable name, then it is recommended to use camel case
02. A CamelCase name being in lower case if there more than one word in the name each subsequent word get an initial cap,creating a hump

  • var userResponse
  • var userResponseTime
  • var response
this style is naming a variable is called camel case.
