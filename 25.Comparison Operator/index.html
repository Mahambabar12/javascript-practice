Comparison Operator

Comparison Operator are used in logical statements to determine equality of different b/w variable or value. they will result in true or false only

  • == // equal to
  • === // equal value to equal type
  • != // Not equal
  • !== // Not equal value or not equal type
  • > // Greater than
  • < // less than
  • >= // Greater than and equal to
  • <= // less than and equal to
