Value conversion with Logical Operator (Optional)

01.Value conversion also work with logical operator && and ||
02.For &&

  • a.If all values are converted to true. it will return last value
  • b.If any of the values converted to false it will that value

03.For ||
  • a. If all values converted to false .it will return last value
  • b. If any value converted to true it will return that value
