For Loop

1. looping in programming language is a feature which facilitates the executions of a set of instructions/ functions repeatedly while same conditions evaluates to true
02. The for statement creates a loop that is executed as a conditions is true
03.It will only stop when the condition become false




  • for (initialization; condition; expression {
    code to bi executed
    }

01. Initialization is done(on time) before the executions of the code block
02.Condition for execution the code block and exit loop
03. Expression is execution (every time) after the code block has been executed



Example

for (let i = 5; i <= 8; i++) {
console.log(i);
}
output
5
6
7
8
